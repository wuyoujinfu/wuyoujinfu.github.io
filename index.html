Checklist

贷款材料清单

提前准备好这些材料，可以大幅提高贷款审批效率和通过率。

🪪 一、身份证明

  • 身份证 — 原件+复印件，有效期内
  • 户口本 — 首页+本人页（部分地区需要）
  • 结婚证/离婚证 — 已婚或离异需提供
  • 居住证明 — 近3个月水电煤账单或房屋租赁合同

💰 二、收入证明

  • 银行流水 — 近6-12个月工资卡流水，银行柜台打印并盖章
  • 单位收入证明 — 公司开具+盖公章，注明月收入、职务、入职时间
  • 劳动合同 — 复印件，证明雇佣关系
  • 个税完税证明 — 个人所得税APP导出或税务局打印
  • 社保/公积金缴存记录 — 人社局或公积金中心查询打印

📊 三、征信与资产

  • 个人征信报告 — 人民银行征信中心官网下载（简版）或柜台打印（详版）
  • 房产证 — 如有，可作为资产证明加分
  • 车辆登记证 — 如有，可作为资产证明加分
  • 定期存单/理财证明 — 银行或券商出具

📎 四、专项贷款补充材料

  • 房贷：购房合同 + 首付款凭证 + 房屋评估报告
  • 车贷：购车合同 + 首付款收据 + 车辆合格证
  • 装修贷：装修合同 + 房产证（证明所有权）
  • 教育贷：录取通知书或学费单据
  • 信用贷：通常仅需身份证 + 收入证明 + 征信报告

📜 一、主体资格

  • 营业执照 — 正副本原件+复印件，已年检
  • 开户许可证 — 银行基本户开户证明
  • 公司章程 — 工商备案版本
  • 股东会/董事会决议 — 同意申请贷款的正式决议文件
  • 法定代表人身份证 — 原件+复印件，有效期内

📈 二、经营证明

  • 经营场所证明 — 房产证或租赁合同 + 近3个月租金收据
  • 经营流水 — 近12个月对公账户银行流水 + 个人经营账户流水
  • 购销合同 — 上下游合同（采购+销售），证明业务真实性
  • 纳税证明 — 近2年完税证明 + 纳税评级证书
  • 增值税发票汇总 — 近6-12个月开票记录
  • 水电费单据 — 近3个月（证明实际经营状态）

📊 三、财务报表

  • 审计报告 — 近2-3年，会计师事务所出具
  • 财务报表 — 资产负债表 + 利润表 + 现金流量表（近2年）
  • 科目明细 — 应收账款、存货、固定资产等（近6个月）
  • 纳税申报表 — 增值税 + 所得税申报表

🏭 四、资产与担保

  • 企业名下房产/土地证 — 厂房、办公楼、商铺、工业用地
  • 设备清单及发票 — 生产设备、运输车辆等固定资产
  • 法人/股东个人房产 — 用于抵押增信
  • 第三方担保函 — 担保公司或其他企业的担保承诺
  • 存单/理财质押 — 定期存单、国债等可质押资产

🔖 五、特定贷款补充材料

  • 税贷：近2年纳税记录 + 纳税评级证书
  • 发票贷：近12个月增值税开票数据
  • 供应链金融：核心企业合作协议 + 应收账款确权文件
  • 项目贷款：项目可行性报告 + 政府批文 + 环评报告
  • 科技贷：高新技术企业证书 + 专利证书 + 研发投入证明

贷款知识指南

了解贷款基础知识，做出更明智的借贷决策。

什么是个人征信报告？▼
个人征信报告是由中国人民银行征信中心出具的记录个人信用信息的文件，包含个人基本信息、信贷记录、公共记录、查询记录等。银行在审批贷款时，会将征信报告作为最重要的参考依据之一。保持良好的征信记录，是获得低利率贷款的关键。
如何查询自己的征信报告？▼
您可以通过以下方式查询：1）中国人民银行征信中心官网（www.pbccrc.org.cn）在线查询；2）前往当地人民银行分支机构柜台查询；3）部分商业银行网点自助查询机。每人每年可免费查询2次，建议每年至少查询1次以了解自身信用状况。
信用评分低怎么办？▼
提高信用评分可从以下方面入手：1）按时还款，避免逾期；2）控制信用卡使用率在70%以下；3）不要频繁申请贷款或信用卡；4）保持稳定的工作和居住信息；5）适当使用信贷产品建立信用记录。通常需要6-12个月才能明显改善。
贷款申请被拒的常见原因？▼
常见原因包括：征信有逾期记录、负债率过高（月供超月收入50%）、收入不稳定或收入证明不足、就业类型不符合产品要求、征信查询次数过多（近期频繁申请）、年龄超出产品范围、贷款用途不合规等。被拒后建议等待3-6个月再尝试。
等额本息 vs 等额本金怎么选？▼
等额本息每月还款金额固定，前期利息占比高，适合收入稳定的人群；等额本金每月还款金额递减，总利息较少，但前期压力大，适合当前收入较高的人群。一般来说，长期贷款（如房贷）可根据自身现金流预期选择，短期贷款差异不大。
通过贷款中介划算吗？▼
贷款中介通常会收取贷款金额1%-5%的服务费。实际上，大部分银行的贷款产品信息都是公开透明的，您完全可以通过自行了解、对比来选择适合的产品。使用无游金服这样的工具，可以在不花费任何费用的情况下，快速了解自己能申请哪些银行的哪款产品。
